# Домашний бар

Телефонно-ориентированное веб-приложение на Expo и React Native Web, которое помогает выбрать коктейль из того, что уже есть дома. Отмечаешь бутылки, соки, цитрус и сиропы, а приложение показывает, что можно смешать прямо сейчас и что выгоднее докупить.

## Возможности

- подбор коктейлей по ингредиентам домашнего бара;
- быстрые сценарии для вечера: без заморочек, свежее, покрепче;
- фильтр по настроению: освежающее, сладкое, кислое, крепкое, горькое;
- поиск по ингредиентам;
- рейтинг совпадений с количеством недостающих ингредиентов;
- раскрываемые карточки с рецептом, бокалом, составом, шагами и подачей;
- подсказки, какие 1-2 ингредиента докупить, чтобы открыть больше коктейлей;
- сохранение выбранного домашнего бара между запусками приложения.

## Текущий объем базы

- 72 коктейля;
- 93 ингредиента;
- 279 связей рецепт-ингредиент;
- 18 стартовых ингредиентов для первого запуска.

Данные генерируются из TheCocktailDB и дополнительного curated-списка классических коктейлей.

## Технологии

- Expo SDK 54;
- React 19;
- React Native 0.81;
- TypeScript в strict mode;
- `@react-native-async-storage/async-storage` для локального сохранения бара.

## Запуск

Установить зависимости:

```bash
npm install
```

Запустить Expo:

```bash
npm start
```

Запустить web-версию:

```bash
npm run web
```

Собрать статическую web-версию для nginx:

```bash
npm run build:web
```

Проверить TypeScript:

```bash
npx tsc --noEmit
```

Перегенерировать базу коктейлей:

```bash
npm run import:cocktails
```
